O que é um commit?
Empacotamento das alterações realizada no projeto.

O que é repositório?
É a pasta do projeto.

O que é pull?
Ele puxa o repositório remoto para o local.

O que é merger?
Mesclagem de duas ou mais branches.

O que é clone?
É uma cópia do repositório remoto.

O que é brach?
É uma ramificação de um repositório.

O que é pull request?
É uma solicitação de uma alteração.

O que é fork?
Ele é um copia de um repositório terciro.

O que é um VCS?
software controle de versão.
